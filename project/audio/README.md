# Audio Assets

This folder contains all audio files for Ironcore Arena.

## Structure

```
audio/
├── sfx/           # Sound effects
│   ├── ui/        # UI sounds
│   ├── weapon/    # Weapon sounds
│   └── battle/    # Battle sounds
└── music/         # Music tracks
```

- 44.1kHz, 16-bit
- Normalized to -6dB
