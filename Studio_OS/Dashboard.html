Dashboard Help

🎯 What's This?

This dashboard is your command center for Ironcore Arena development. It tracks tasks, shows progress, and helps you decide what to work on next.

📋 Task Management

  • Active Tasks: Click any task to open its file
  • Next Up: Shows the highest efficiency task based on impact/effort ratio
  • Completed: Recently finished work with timestamps

🚀 Next Up Selection

The "Next Up" task is chosen by:

  • High impact, low effort tasks first
  • Unblocking dependencies
  • Resource availability (Kimi/Agent Swarm/etc.)
  • Current project priorities

💡 Economics Strategy

All executors are subscription-based (no marginal cost):

  • Kimi 2.5: Blueprints, local work, quick fixes
  • Agent Swarm: 10+ files, parallel execution
  • Deep Research: Complex research tasks
  • Codex: Large refactoring projects

⌨️ Shortcuts

  • Ctrl/Cmd + R - Refresh data
  • ? - Show this help
  • Esc - Close modals

📝 Updating Data

This dashboard auto-updates when tasks are completed. To manually update, click the Refresh button or press Ctrl+R.
